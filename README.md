# 项目用途
根据天天基金网进行(http://fund.eastmoney.com/allfund.html)基金收益排行统计

#### NOTE:
- 丢弃货币基金提高效率。
- 只收集基金最近一月的收益，因为时间太久不具参考性。
- 丢弃最近一个月收益率低于百分之一的基金，提高效率。
- 统计结果按照基金开头数字统计，会在目录下生成一个fund_log.log文件存放统计结果。
- 统计结果从高到底只列出前十名。
